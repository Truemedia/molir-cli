# Molir CLI
[Molir](https://www.npmjs.com/package/molir) (A.k.a Most Likely/Obvious Intent Recogniser) as command line tool

## Installation

NPM
```bash
    npm i -g molir-cli
```

Yarn
```bash
    yarn global molir-cli
```

## Usage
Bash
```bash
    molir-cli --score=0.75 --i=sample_intents.json --t='Whats todays news?'
```
